# 🏓 Pong Game

A classic **2-Player Pong Game** built using **Python Turtle Graphics** with **Pygame** for sound effects. This project recreates the famous arcade game where two players compete by bouncing the ball back and forth while trying to score points.

## ✨ Features

- 🎮 Two-player gameplay
- ⌨️ Keyboard controls
- ⚡ Smooth ball movement
- 🔊 Paddle collision sound effects using Pygame
- 📊 Live score tracking
- 🔄 Ball resets after each point
- 🖥️ Built using Python Turtle Graphics

## 🛠️ Technologies Used

## 🎮 Controls

### Player A (Left Paddle)

| Key | Action |
|-----|--------|
| W | Move Up |
| S | Move Down |

### Player B (Right Paddle)

| Key | Action |
|-----|--------|
| ↑ Up Arrow | Move Up |
| ↓ Down Arrow | Move Down |

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/pong-game.git
```

### Go to the project folder

```bash
cd pong-game
```

### Install Pygame

```bash
pip install pygame
```

### Run the game

```bash
python main.py
```

---

## 🎯 Game Rules

- The ball starts from the center of the screen.
- Players use their paddles to hit the ball.
- If a player misses the ball, the opponent scores one point.
- The ball resets after each point.
- The game continues until you decide to close it.
